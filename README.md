# observabilidade

Projeto academico desenvolvido na disciplina de Observabilidade da Pos-graduacao na Unipe. Explora coleta de metricas, logs e rastreamento em aplicacoes Java.

---

## Tecnologias

- Java
- Spring Boot
- Micrometer
- Docker

---

## Pre-requisitos

- Java 11+
- Maven
- Docker (opcional)

---

## Instalacao

```bash
git clone https://github.com/fabiodocarmo/observabilidade.git
cd observabilidade
```

---

## Como usar

```bash
./mvnw spring-boot:run
```

Consulte os endpoints de actuator para visualizar metricas e informacoes de saude da aplicacao.
